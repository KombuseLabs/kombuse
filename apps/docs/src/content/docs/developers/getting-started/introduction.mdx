Kombuse is a TypeScript monorepo built with Turborepo and Bun.

## Architecture

The project is organized into apps and packages:

### Apps

- **desktop** — Electron desktop application
- **web** — Vite + React Router web application
- **docs** — This documentation site (Astro Starlight)
- **server** — Fastify REST API server

### Packages

- **agent** — Agent backend implementations (Claude Code, Codex, etc.)
- **core** — Command registry and keybindings (framework-agnostic)
- **mcp** — MCP server and tool definitions
- **persistence** — Database layer (SQLite, repositories, migrations)
- **pkg** — Package/build utilities
- **services** — Business logic services
- **types** — Shared TypeScript types
- **ui** — React component library (shadcn/ui + Radix)

## Tech Stack

- **Runtime:** Bun, Node ≥18
- **Framework:** Vite, React 19, React Router 7, Fastify 5
- **Styling:** Tailwind CSS 4 with dark mode
- **Components:** shadcn/ui + Radix UI primitives
- **Language:** TypeScript 5.9 (strict mode)
